Perfect Picnic Spot

Create a function named decidePicnicSpot that receives lovesShade and lovesFlowers as parameters.

This function helps decide the perfect picnic spot based on preferences for shade and flowers:

If lovesShade is true and lovesFlowers is false, return "Oak tree".
If lovesShade is false and lovesFlowers is true, return "Flower bed".
If both are true, return "Either spot is perfect!".
If both are false, return "Either spot is perfect!".
Parameters:

lovesShade (boolean): True if you prefer shade, False otherwise.
lovesFlowers (boolean): True if you enjoy flowers, False otherwise.
The function returns a String representing the perfect picnic spot based on your preferences.
